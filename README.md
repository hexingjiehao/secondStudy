# 7.列表
无序列表：使用-+*都可以。注意：- + * 跟内容之间都要有一个空格
- 列表内容
+ 列表内容
* 列表内容

有序列表：数字加点。注意：序号跟内容之间要有空格
1. 列表内容
2. 列表内容
3. 列表内容

列表嵌套：上下级之间敲3个空格
1. 一级有序列表
   * 二级无序列表

# 9.代码：
单行代码使用反引号包起来：`代码内容`

代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行
```
  代码...
  代码...
  代码...
```
