1.学习PKI知识点：
    1.原文参考：https://www.cnblogs.com/jerain6312/p/8572841.html
    2.名字：Public Key Infrastructure（PKI)，中文叫做公开密钥基础设施，也就是利用公开密钥机制建立起来的基础设施
    3.核心内容：
        1. PKI的核心是身份证明书的发行
            答：为了验证网络上进行交流的人的身份是否是其本人，需要一个权威机构(CA)给每个人发放一个证明书--数字证书
                还需要一个统一管理证明书的档案库。防止私人伪造证明书

        2. 构成PKI的要素只有三个
            答：证明书：存放在硬盘上，拥有X.509的协议格式
                认证机关：制作证明书，本质是一个网络应用程序
                证书库：分发证明书，本质是一个文件系统

                PKI指的是证明书的制作和分发的一种机制。在这个机制的保障前提下，进行可信赖的网络通信

        3. 证明书里面的密钥
            答：证明书里有一个代表个人信息的公共密钥，将你要发送给比尔的内容进行加密，然后发送给比尔。
                该加密内容只能用比尔自己才有的另一个“私人的密钥”才能解密。其他人无法解密

                PKI提供的证明书可以用来 身份确认 和 通信加密。同时实现了这两个重要的功能。

        4. 什么是“公开密钥”，什么是“私有密钥”
            答：放在“证明书里面的密钥”可以被任意自由分发，这里的“证明书里的密钥”被叫做“公开密钥（Public Key）”。
               与此相对，本人保管的那个“私人的密钥”就要做“私有密钥（Private Key）”。

                公开密钥是放在证明书里面的，所有用什么样的方式去分发证明书都没有关系。
                放到U盘里给别人，或者放到网上让人任意下载，或者用邮件发送，都是可以的。

        5. “拿什么去信任你？我的证明书”
            答：认证机关所用的证书生成器说到底也就是一个软件而已，如果搞到这个软件，谁都可以发行证明书。
                所以说，在技术上，伪造证明书是非常简单的

                认证机关就至关重要了，认证机关的可信度，直接与证书的可信度挂钩，也就是与整个PKI机制的可信度息息相关。
                其实是一个社会基础设施建设的话题了。认证机关都是由政府在主导建设
